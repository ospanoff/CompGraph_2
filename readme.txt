Оспанов Аят, ММП, 317 группа. 2014 год, Москва.

Были сделаны все доп. задания. Но задание "Пирамида дескрипторов" (9.3) была немного видоизменена. Вместо того, чтобы считать HOG для одной картинки и далее для 4х четвертинок, считался HOG с одним количеством клеток, затем с другим, более измельченным и т.д. Допустимость такого решения была обсуждена и подтверждена на форуме.

Также были добавлены скрипты для обучения, классификации и оценки точности для двух выборок:
1) train.sh - обучить классификатор на выборке binary
2) test.sh - классифицировать изображения из выборки binary
3) cmp.sh - оценить точность классификатора для выборки binary
4) train_m.sh - обучить классификатор на выборке multiclass
5) test_m.sh - классифицировать изображения из выборки multiclass
6) cmp_m.sh - оценить точность классификатора для выборки multiclass

Файлы с обученными моделями хранятся в корневой папке
1) model_binary.txt - модель для выборки binary
2) model_multiclass.txt - модель для выборки multiclass
